# Aura Portfolio React Conversion

This is the React + Vite + Tailwind CSS version of the original HTML/CSS portfolio.

## Run locally

```bash
npm install
npm run dev
```

## Assets

Put these files into `src/assets/` if you have them:

- `aura.jpg`
- `cat-construction.jpg`
- project images

Then import them inside the relevant component/data file.
